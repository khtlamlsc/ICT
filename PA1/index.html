Super Router Sales Decision Maker

Task Requirement

You are a junior developer at a major electronics retailer. Your manager wants a simple tool to help the sales team recommend the right kind of router to customers. Your program will act as an expert advisor, asking customers a few simple questions to determine the best router for their needs based on the provided magazine article. The key challenge is to implement this logic using simple conditional statements and support multiple languages to assist a diverse customer base. The tool needs to be straightforward, ask for the customer's internet plan speed, and then suggest a category of router.

Resources

Read Magazine in eLibrary Download Skeleton Code

Objectives

  • Create a python program.
  • Analyze requirements from magazine article provided by eLibrary.
  • Design and implement a decision-making process using conditional logic (`if-else`).
  • Handle user input and perform basic data parsing.
  • Structure a program that supports translation using AI.

Decision Logic

Your program should use the following logic, derived from the magazine article:

  • If ISP speed is ≤ ___________ Gbps, recommend Wi-Fi 6:
    • If user wants _________________ → Recommend _____________________.
    • If user wants _________________ → Recommend Asus RT-AX86U.
    • If user wants _________________ → Recommend _____________________.
  • If ISP speed is > ___________ Gbps, recommend Wi-Fi 7:
    • If user wants ________________ → Recommend Netgear Nighthawk RS300.
    • If user wants ________________ → Recommend _____________________.
    • If user wants ________________ → Recommend _____________________.

Program Requirements

Your program must follow this specific flow:

  1. Language Selection: First, ask the user for their preferred language (English, 中文, 日本語, 한국어).
  2. Product Inquiry: Ask what product they want to buy. The program should only proceed if they enter the word for "Router" in their chosen language.
  3. Speed Analysis: Ask for their ISP's theoretical internet speed in Gbps. You will need to parse the numerical value from their input (e.g., get `9.5` from `"9.5 Gbps"`).
  4. Primary Recommendation: Based on the speed, use an `if-else` statement to recommend either a Wi-Fi 6 or Wi-Fi 7 router category.
  5. Detailed Recommendation: After the category is chosen, ask a follow-up question to determine the user's primary need (e.g., budget, gaming, or coverage).
  6. Final Recommendation: Based on their final answer, recommend one specific, award-winning router model from the magazine article.

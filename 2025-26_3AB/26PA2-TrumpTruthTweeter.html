The program will ask the user to choose a country to make a deal with and then display a unique outcome based on their choice.

  1. 1️⃣ Get User Input: Ask the user to enter a country name. Use `.capitalize()` to ensure "china" becomes "China" for easier checking.

    2. Handling User Input

    To make your `if` statements work reliably, it's good practice to format the user's input. The `.capitalize()` method turns the first letter of a string into uppercase and the rest to lowercase, so `CHINA`, `china`, and `cHiNa` all become `China`.

    country_input = input("Choose a country to make a deal with: ")
country = country_input.capitalize()
  2. 1️⃣ Decision Making (`if-elif-else`):
    • If the country is "China", print: `Outcome: This is the worst trade deal in the history of trade deals.`
    • Else if the country is "Russia", print: `Outcome: A perfect phone call. Tremendous friends.`
    • Else if the country is "Ukraine", print: `Outcome: I would like you to do us a favor though...`
    • Else (for any other country), print: `Outcome: They love me there. The best deal.`

    3. The `if-elif-else` Structure

    This structure lets your program check multiple conditions in order. It runs the code for the first condition that is `True` and skips the rest.

    if country == "China":
    print("Specific outcome for China.")
elif country == "Russia":
    print("Specific outcome for Russia.")
else:
    print("A default outcome for all other countries.")
